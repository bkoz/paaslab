IT'S SO EASY

Goal: Stand up WordPress in the Cloud in record time.

  1. Create a php application (rhc app create myblog -t php-5.3 -p password).
  2. Add a mysql cartridge (rhc cartridge add -c mysql-5.1 -a myblog -p password).
  3. cd myblog
  4. Clone an OpenShift Quickstart from github:(instructions also on github)
    • git remote add upstream -m master git://github.com/openshift/wordpress-example.git
    • git pull -s recursive -X theirs upstream master
    • git push
  5. Now visit the URL that was created
