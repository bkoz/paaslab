DIY CARTRIDGE

Goal: Imbed an application into a cartridge

  1. Create a diy application
  2. Download a sample binary
  3. unzip the sample app into the top level directory of your app
  4. Have a look at start and stop files in the .openshift/action_hooks directory
  5. Using git, add and commit the new files, then push.
