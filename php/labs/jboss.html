JBoss Developer Studio(JBDS)/OpenShift Lab

Goal: To create a JBoss application with JBDS and run it on OpenShift.

If all this GIT and CLI stuff is not your style checkout JDBS!

JBDS Requires a => v1.6 JRE

Download and install JBDS
(instructor may have a local download area)

The Big Picture

  1. Create a JBDS Project
  2. Create an OpenShift Application
  3. Link the two together
  4. Publish (push) your application to OpenShift
The Details.
  1. Start JBDS. In the Project Examples Tab open the JBoss Quickstarts tab and choose the kitchensink, helloworld or greeter example.
  2. Optionally, run the app locally.
  3. Create a new OpenShift Application from JBoss Central tab (use a jbosseap cartridge).
  4. Uncheck create a new project and browse for an existing JBDS project.
  5. When this wizard finshes an OpenShift app was created but your local contents have not been published.
  6. Open the Server tab near the bottom and expand your openshift application.
  7. Right click on your project and choose 'full publish'.
  8. The first publish will take some time for the maven artifacts to be downloaded.
  9. Now make some changes to the app and republish
