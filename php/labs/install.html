Software Installation and Configuration Lab

Goal: To install client tools for OpenShift.

We will be using 2 pieces of software, Ruby and GIT. A Ruby gem called rhc is used to interact with the OpenShift broker to create applications, add cartridges and obtain information. For revision control, we will be using git to push and pull application source code to and from the OpenShift nodes.

If you prefer not to install git and ruby, you can log in to broker.example.com using ssh (putty on Windows) and skip to the rhc setup step.

Quick Instructions

Download and install ruby, then git for your platform.

Open a terminal shell (command prompt on Windows) and install rhc.

C:> gem install rhc

Open a terminal shell (git shell on Windows) and run the following command: (The instructor will provide everyone with a unique OpenShift user name and password)

$ rhc setup --server broker.example.com

Important: When prompted for your namespace, set it to something unique, like your user name.

If everything goes well the final message should resemble:

Your client tools are now configured.
