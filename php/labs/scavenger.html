ssh SCAVENGER HUNT

Goal: Hunt for buried OpenShift treasures.

  1. Choose an app that you've created. What public IP address is using? (hint: host or dig)
  2. Using the web console or CLI tools, find out the ssh login string for an application. (hint: rhc domain show)
  3. Now, ssh into the application.
  4. What is your home directory? (hint: pwd)
  5. Try to list all home directories. (hint: ls)
  6. What private IP addresses is your app using? (hint: env)
  7. How much memory is this app using? (hint: oo-cgroup-read memory.usage_in_bytes)
  8. Stop your app and check the memory usage (hint: ctl_app graceful-stop)
  9. Restart your app (hint: ctl_app start)
  10. What is the total memory available for your app? (hint: oo-cgroup-read memory.limit_in_bytes)
