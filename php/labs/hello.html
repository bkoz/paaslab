HELLO OPENSHIFT

Goal: To get familair with OpenShift's command line interface

So this is the ice breaker example that you've all been waiting for. We will create a simple php web application, publish it, make some changes and republish. It shoud give you a good feel for the developer workflow using OpenShift's command line tools.

First, an OpenShift URL primer

  1. Create a php application. (rhc app create myphp php-5.3)
  2. Now visit the URL that was created
  3. A local git repo was created, change into that directory.
  4. edit php/index.php and make a code change and save your work.
  5. git commit -am "Some informative message"
  6. git push
  7. Now revisit your app.
