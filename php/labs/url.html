A bit about OpenShift URLs

When an OpenShift application is created, a DNS record is created based on the application name, namespace and domain.

OpenShift URL format

http://applicationName-namespace.domain

In this workshop, our domain is example.com and we will set our namespace = OpenShift login.

Example URL:

http://myjboss-demo.example.com
